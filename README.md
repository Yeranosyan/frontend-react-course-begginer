## 🚀 Getting Started

```bash
# Clone the project
git clone https://github.com/Yeranosyan/frontend-react-course-begginer.git

# Navigate into the folder
cd react-todo-ep1

# Install dependencies
npm install

# Start development server
npm run dev
```
